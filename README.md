# Honeycomb

OSS 活動への参加を支援するために作られた日本人向けのプロジェクトです。

ハチの巣をみんなで作りながら、Git と GitHub の使い方を学びましょう。

## 参加方法

Git講座のチュートリアルに沿って、あなたのセルをハチの巣に追加してみてください。

[https://gitnote.rubydog.jp](https://gitnote.rubydog.jp)

## データ構造

`data/cells.json` にハチの巣のセルデータが格納されています。

```json
{
  "address": 0,
  "github_id": "rbdog",
  "color": "#C70067",
  "text": "ルビードッグ"
}
```

まだ誰も使っていない `null` の項目を選んで編集し、プルリクエストを送ってください。
